# Hedon charset

**Keyboard charset** [a-zA-Z] ! " # $ % & ' ( ) - = ^ ~ \ | @ ` [ ] { } + ; * : < > , . ? / _

**Hedon charset**
- [a-zA-Z] alphabet
- [0-9] numeric
- @ fetch
- ! store
- . dot
- \+ plus
- \- of
- " str
- [ begin-quot
- ] end-quot
- : begin-def
- ; end-def

**Not used**
- \# $ % & ' ( ) = ^ ~ \ | ` { } * < > , . ? / _

**Hedon word compared to C operator**
- \+ \+
- \- \-
- \* mul
- / div
- % mod
- | bor
- & band
- \<< shl
- \>> shr
- \< le
- \> gr
- \<= leq
- \>= grq
- \* @
