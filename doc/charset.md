# Hedon charset

**Keyboard charset** [a-zA-Z] ! " # $ % & ' ( ) - = ^ ~ \ | @ ` [ ] { } + ; * : < > , . ? / _

**Hedon charset**
- [a-zA-Z] alphabet
- @ fetch
- ! store
- . dot
- , comma
- + plus
- \- of
- * asterisk
- / slash

**Hedon word compared to C operator**
- \+ \+
- \- \-
- \* \*
- / /
- % mod
- | bor
- & band
- \<< shl
- \>> shr
- \< le
- \> gr
- &. ref (addr)
- *. de
- -. neg
